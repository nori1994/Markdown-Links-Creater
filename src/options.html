Setting

Set wheter to stock Markdown links to ClipBoard.

ONにしている間にリンクを作成した場合、ストックしたリンクとして保存されます。 OFFの時にリンクを作成してもストックしたリンク一覧に追加はされません。

ONからOFFにした時は、最近作成したリンクがクリップボードにコピーされています。ストックされているリンクは勝手に削除されません。
