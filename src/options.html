Setting

Choose created link to stock on “Stocked Links” or not.

ON: When you do “Create Markdown Links,” the link just created is added to the bottom of “Stocked Links” (below list) then all links on “Stocked Links” copy to ClipBoard.

OFF: When you do “Create Markdown Links,” only the link just created copy to ClipBoard.

When switching from ON to OFF, only the most recently created link remains as stocked link, and the other links disappear.

Stocked Links

Edit the “Stocked Links” to choose which links copying to ClipBoard.

If you push the "Copy to ClipBoard" button, all links on “Stocked Links” copy to ClipBoard.
